#Inicio do arquivo em Python para poder ser adicionado no GitHub.

pip freeze > requirements.txt (para criar o arquivo de itens requeridos)
git init (Inicia o git/transforma em repositório git)
git add README.md (Adicionado um arquivo README)
git status (Para ver todos os arquivos)
git add -A (Adiciona todos os arquivos para poder lançar)
git commit -m "Primeiro Commit" (Faz commit dos arquivos)
git branch -M main (define o nome da branch principal como main)
git remote add origin https://github.com/brendowsabe-spec/Miniprojeto_Brendow_De_Souza_Adriano_Analise_de_Dados_T1.git (Conecta o arquivo local ao repositório do GitHub)
git push - u origin main (envia os arquivos para o GitHub)

Depois apenas repetição dos comandos:
git add .
git commit -m "Mensagem"
git push
