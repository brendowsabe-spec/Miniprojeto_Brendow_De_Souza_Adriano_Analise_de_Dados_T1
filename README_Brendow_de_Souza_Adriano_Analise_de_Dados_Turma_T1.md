# Inicio do arquivo em Python para poder ser adicionado no GitHub.
1. pip freeze > requirements.txt (para criar o arquivo de itens requeridos)
2. git init (Inicia o git/transforma em repositório git)
3. git add README.md (Adicionado um arquivo README)
4. git status (Para ver todos os arquivos)
5. git add -A (Adiciona todos os arquivos para poder lançar)
6. git commit -m "Primeiro Commit" (Faz commit dos arquivos)
7. git branch -M main (define o nome da branch principal como main)
8. git remote add origin https://github.com/brendowsabe-spec/Miniprojeto_Brendow_De_Souza_Adriano_Analise_de_Dados_T1.git (Conecta o arquivo local ao  repositório do GitHub)
9. git push - u origin main (envia os arquivos para o GitHub)
